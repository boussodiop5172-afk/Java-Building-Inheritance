# Building Inheritance Application

This Java application is designed for an object-oriented programming assignment. It demonstrates key OOP concepts like class inheritance, data encapsulation, and user input processing.

## Project Structure
* **Building.java (Parent Class):** Contains private attributes for `siding` and `squarefeet` with public getters and setters.
* **House.java (Child Class):** Inherits from Building; calculates total square feet based on individual home rooms.
* **Commercial.java (Child Class):** Inherits from Building; calculates total square feet based on 3 commercial rental rooms.
* **MainApp.java:** The driver class that interacts with the user via console input and displays the final results.
